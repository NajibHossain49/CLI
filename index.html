Tailwind CLI Installation Steps

Initialize your project:

Run the following command to create a package.json file with default settings:

  1. npm init -y

Install Tailwind CSS:

Run the following command to Install Tailwind CSS and its dependencies:

  1. npm install -D tailwindcss

Generate the Tailwind configuration file:

Run the command to create a tailwind.config.js file:

  1. npx tailwindcss init

Tailwind CSS Configuration:

Copy the following code,Clear the previous codes and replace it into your tailwind.config.jsfile: 

      
/** @type {import('tailwindcss').Config} */
module.exports = {
content: ["./src/**/*.{html,js}"],
theme: {
  extend: {},
},
plugins: [],
}

Create your CSS file:

Create a file (e.g., src/input.css) and copy the following code and paste it into your src/input.css file: 

        
@tailwind base;
@tailwind components;
@tailwind utilities;

Build your CSS:

Use the following command to build your CSS and watch for changes:

  1. npx tailwindcss -i ./src/input.css -o ./src/output.css --watch

Add the build script to package.json:

Replace the following script to your package.json to automate the build process:

    
"scripts": {
  "build:css": "npx tailwindcss -i ./src/input.css -o ./src/output.css --watch"
}

Create index.html file:

Create an index.html file in your src/index.html directory and include the following line to link your compiled CSS:

        
<link href="./output.css" rel="stylesheet">

Run the build script:

Execute the following command to start the Tailwind CSS build process:

  1. npm run build:css

Note ❗: All steps should be performed in the VS Code terminal.You must install Node.js to install Tailwind CSS CLI. If Node.js is not installed, click here to install Node.js.

Note ❗: After configuring all steps successfully, Tailwind CSS should be applied. If it does not work, run npm run build:css. Additionally, when you restart VS Code or Close VS Code even if Kill the Terminal, make sure to run the npm run build:css once again to ensure everything works correctly.
